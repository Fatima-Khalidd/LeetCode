Intuition

Find the length of the last word in a string.

Approach

1-Trim the string to remove leading and trailing whitespace.
2-Iterate backwards through the string.
3-Count characters until a whitespace or the beginning of the string is encountered.
4-Return the count as the length of the last word.

Complexity

  • Time complexity:
    O(n), where n is the length of the input string.

  • Space complexity:
    O(1), using only constant extra space.

Code

class Solution {
    public int lengthOfLastWord(String s) {
        s=s.trim();
        int length=0;
        boolean WordFound=false;
        for(int i = s.length()-1;i>=0;i--)
        {
            char c=s.charAt(i);
            if(c!=' ')
            {
                length++;
                WordFound=true;
            }
            else
            break;
        }
        return length;
    
}
    }
